# API Documentation

> **Warning**
>
> This is documentation for the **BILLZ 2.0** API.
>
> Documentation for the BILLZ 1.0 API is available at https://api.billz.uz/docs/#/
>
> BILLZ 1.0 and BILLZ 2.0 are independent, unrelated systems.

The BILLZ 2.0 REST API interface is used to integrate eCommerce systems with the BILLZ 2.0 APP. All API access is made over HTTPS and is available at https://api-admin.billz.ai. All data is sent and received in JSON format.
