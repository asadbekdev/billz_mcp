# How to Create an Integration Key

1. Go to **Settings** and select **Company Settings**.
2. Navigate to **Integration Keys**.
3. Click the **New Key** button and enter a **key name** — this can be the name of your integration or any identifier you prefer.
4. Find the key in the list and copy it to the clipboard.
