# Как создать ключ интеграции

1. *зайдите в раздел **Настройки** и выберете **Настройки компании***
2. *перейдите в пункт **Ключи интеграции***
3. *нажмите на кнопку **Новый ключ** и ведите **название ключа.**  Это может быть название интеграции, или любой идентификатор*
4. *найдите ключ в списке и скопируйте в буфер обмена*
